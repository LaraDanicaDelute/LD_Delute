Brief History

In the 90's therewere various of scripting languages and developers struggled to
support each and apply to their web, to bring stability Netscape submitted
JavaScript to ECMA international to define standards to be implemented
and shared with other browser vendors, so they released ECMA Script in June of 1997.






What is ECMA Script

ECMA script is a standard that scripting languages such as JavaScript follows,
it is to asure the interoperability of webpages between different browsers.
